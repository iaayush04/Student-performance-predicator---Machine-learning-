# Student-performance-predicator---Machine-learning-
Student Performance Prediction is a machine learning project designed to analyze academic and behavioral factors to predict students' academic performance. The system uses historical educational data and predictive algorithms to identify patterns that influence student success.

Early prediction of student performance can help educators and institutions provide timely academic support to students who may be at risk of poor performance.

This project demonstrates the application of data science and machine learning techniques to solve real-world problems in the education sector.

# Problem-Statement
Educational institutions often struggle to identify students who may 
perform poorly in advance. Early prediction of student performance 
can help teachers and institutions provide additional support.

This project uses machine learning algorithms to analyze various 
student attributes such as study time, attendance, and previous 
grades to predict academic performance.

# Project Objectives
• Analyze student academic data
• Build a predictive machine learning model
• Identify factors affecting student performance
• Provide early insights for academic improvement

#  Dataset

Dataset includes the following features:

• Study time
• Attendance
• Previous grades
• Parental education
• Internet access
• Extra curricular activities

# Machine Learning Algorithms Used
• Linear Regression
• Random Forest
• Decision Tree
• Logistic Regression
